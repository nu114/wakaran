# 開発サーバーの起動
```
## 方法1: npmスクリプトを使用
npm run develop
## または
npm start

## 方法2: pnpmを使用（このプロジェクトではpnpm-lock.yamlがあるため）
pnpm develop
## または
pnpm start

## 方法3: 直接gatsbyコマンドを実行
npx gatsby develop
```

# 本番用ビルド:
```
npm run build
## または
pnpm build
```

# ビルドしたサイトのプレビュー:
```
npm run serve
## または
pnpm serve
```

# キャッシュのクリア:
```
npm run clean
## または
pnpm clean
```
